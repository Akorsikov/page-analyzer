### Requirements

* PHP >= 8.0
* Composer >= 2.6.6
* PostgreSQL >= 16.4
* GNU Make >= 4.3

### Setup

```bash
git clone https://github.com/Akorsikov/php-project-9.git
cd php-project-9
make setup
```

### Run

Create a database and upload the database.sql file into it.

```bash
make start
```

### Usage

Open in browser: <http://localhost:8080>
